## node-snapshot

This is a work in progress, the main idea is to generate a carrousel with thumbnails of a site. Like facebook when you add an url. 

## State
At this moment you are able to get an array with the images availables in a site.

## Use

    > git clone git://github.com/alejandromg/node-snapshot.git
    > cd node-s*
    > npm install
    > node index.js

Then:
  
    > curl http://localhost:9000/?url=github.com

Output:
    // An array with the images from the site
